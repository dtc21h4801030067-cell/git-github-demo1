Quản lý mã nguồn với Git & GitHub

Dự án mẫu dùng để thực hành repository, commit, branch, merge và Pull Request.

Mục tiêu thực hành

  • Khởi tạo repository bằng Git.
  • Tạo commit có nội dung rõ ràng.
  • Phát triển tính năng trên branch riêng.
  • Push mã nguồn lên GitHub và tạo Pull Request.
  • Thực hành xử lý merge conflict.

Quy trình Git cơ bản

Working Directory → Staging Area → Local Repository → GitHub
